# Inline-Styles-Basic-Interaction-and-Server-Side-Validation

## Objective
To extend HTML forms using inline styles, implement client-side validation using JavaScript, and server-side validation using Node.js and Express.

## Features
- Inline CSS styling
- Client-side validation using JavaScript
- Server-side validation using Express
- Temporary server-side data storage
- Dynamic HTML rendering using EJS

## How to Run
1. Open folder in VS Code
2. Open terminal
3. Run:
   npm install
   npm start
4. Open browser:
   http://localhost:3000

## Output
- Validated form submission
- Error messages for invalid input
- Successfully submitted data displayed on page
